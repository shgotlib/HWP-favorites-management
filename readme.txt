Just another favorite list management plugin. flexible and simple to use.

== Description ==
This plugin allows your users to add articles (or some another custom post type) to a personal reading list/wishlist.
Using the plugin is simple, just add 3 shortcodes and everything is ready.

Show to user how much items he saved by paste this shortcode:
[favorites]

Show to user his favorites list by paste this shortcode:
[favorites_list]

Place a button on single post or single post type for saving the item to user \\ \'s list by paste this shortcode:
[favs-btn]

== Installation ==
Once the plugin installed you can go to: /wp-admin/users.php?page=ff-favorites-settings
